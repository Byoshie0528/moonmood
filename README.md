# moon mood

## 概要
気分アイコンとコメントを記録し、一覧で振り返ることができるWebアプリケーションです。

## 使用技術
・Java（Servlet/JSP）
・HTML / CSS / JavaScript
・MySQL

## 主な機能
・気分とコメントの登録機能
・一覧表示機能
・データの保存機能

## 制作背景
日々の気分を簡単に記録・振り返りできるようにしたいと考え、制作しました。

## 工夫した点
・シンプルで使いやすい画面設計を意識しました 
・データの保存や表示の流れを理解しながら実装しました
・自作の気分アイコンを表示してシンプルながら親しみにあるデザインにしました。
